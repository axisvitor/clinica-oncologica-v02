   Risk: Duplicate quiz responses recorded

8. IDEMPOTENCY MIDDLEWARE RACE
   File: app/middleware/idempotency.py:232
   Problem: Event deduplication check-then-create not atomic
   Risk: Duplicate webhook processing

9. PUBSUB USER ENUMERATION RACE
   File: app/services/redis_pubsub_manager.py:302
   Problem: Iterating dict while other thread modifies it
   Risk: RuntimeError crash, message delivery failure
