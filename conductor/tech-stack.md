# Technology Stack

## # Backend
*   **Language:** Python 3.13+
*   **Web Framework:** FastAPI
*   **Database (ORM):** SQLAlchemy 2.0 with Psycopg 3
*   **Task Queue:** Celery with Redis broker
*   **Validation:** Pydantic v2
*   **Migrations:** Alembic

## # Frontend (Dashboard)
*   **Framework:** React 19
*   **Build Tool:** Vite 6
*   **Styling:** TailwindCSS
*   **State Management:** TanStack Query (React Query)
*   **Components:** Radix UI / shadcn/ui patterns
*   **Language:** TypeScript

## # Quiz Interface
*   **Framework:** Next.js 14 (App Router)
*   **Styling:** TailwindCSS
*   **Validation:** Zod
*   **Language:** TypeScript

## # Infrastructure & Integrations
*   **Authentication:** Firebase Auth (Admin SDK on backend)
*   **Messaging:** WhatsApp via Evolution API
*   **AI Service:** Google Gemini (via LangChain)
*   **Hosting/Deployment:** Railway (Railpack builder)
*   **Monitoring:** Sentry, Prometheus, Flower
*   **Database:** PostgreSQL
*   **Caching/Broker:** Redis 7+
