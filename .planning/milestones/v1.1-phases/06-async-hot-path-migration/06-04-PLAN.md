<objective>
Migrate saga orchestrator sub-components (`compensation.py` and `steps.py`) from sync `Session` to `AsyncSession` for all 8 annotated TODO(async-migration) instances. Update callers in FastAPI routes and Celery tasks.

Purpose: Saga compensation and step execution handle patient onboarding transactions. Sync DB calls during compensation create data integrity risk via timeout -- if the event loop blocks during a compensation rollback, partial state corruption is possible.

Output: `compensation.py` (5 TODOs) and `steps.py` (3 TODOs) use `await self.db.execute(select(...))` and `await self.db.commit()`. FastAPI callers inject `AsyncSession`. Celery caller bridges via `asyncio.run()`.
</objective>
