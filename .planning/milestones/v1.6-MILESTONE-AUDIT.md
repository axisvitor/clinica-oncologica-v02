### Integration Findings

**M-1 (LOW):** `webhook.py:37` reads `WHATSAPP_WUZAPI_WEBHOOK_SECRET` from `os.environ.get()` instead of `settings.WHATSAPP_WUZAPI_WEBHOOK_SECRET`. Both resolve identically at runtime. Inconsistent with the centralized settings pattern used by all other callers. Affects WH-04, CFG-01.

**M-2 (MEDIUM):** `POST /whatsapp/contacts/{instance}/sync` returns HTTP 200 but the background task calls `sync_contacts()` which raises `NotImplementedError`. The endpoint is misleading — should return 501 or be removed. Affects OUT-02.
