   - Has a `rollback()` method that calls the sync savepoint's `rollback()` and returns an awaitable.
   - Has an `is_active` property that delegates to the sync savepoint's `is_active`.
   - Implements `__await__` so `await db.begin_nested()` resolves to the proxy itself.
   - Implements `async __aenter__` / `async __aexit__` for context manager usage.
3. Return the proxy so that `savepoint = await db.begin_nested()` resolves correctly.

Implementation pattern (add INSIDE the SyncToAsyncSessionAdapter class, before `__getattr__`):

```python
def begin_nested(self):
    sync_savepoint = self._sync_session.begin_nested()
    adapter = self

    class _SavepointProxy:
        def __init__(self, sp):
            self._sp = sp

        @property
        def is_active(self):
            return self._sp.is_active

        def commit(self):
            self._sp.commit()
            return adapter._awaitable()

        def rollback(self):
            self._sp.rollback()
            return adapter._awaitable()

        def __await__(self):
            async def _resolve():
                return self
            return _resolve().__await__()

        async def __aenter__(self):
            return self

        async def __aexit__(self, exc_type, exc_val, exc_tb):
            if exc_type is not None and self._sp.is_active:
                self._sp.rollback()
            return False

    return _SavepointProxy(sync_savepoint)
```
