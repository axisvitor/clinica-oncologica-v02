<interfaces>
Standard migration template (apply to each file):

```python
# IMPORTS: Replace
from app.database import get_db
# With:
from app.core.database.async_engine import get_async_db
from sqlalchemy.ext.asyncio import AsyncSession
from sqlalchemy import select

# SIGNATURE: Replace in each handler
db=Depends(get_db)
# With:
db: AsyncSession = Depends(get_async_db)

# READ single:
# db.query(Model).filter(Model.id == id).first()
result = await db.execute(select(Model).where(Model.id == id))
obj = result.scalar_one_or_none()

# READ list with filter:
# db.query(Model).filter(...).all()
result = await db.execute(select(Model).where(...))
items = result.scalars().all()

# WRITE: Add await
db.add(obj)           # stays sync
await db.commit()     # was db.commit()
await db.refresh(obj) # was db.refresh(obj)
await db.delete(obj)  # was db.delete(obj)
await db.rollback()   # was db.rollback()
```

For `template_admin.py` which queries two models:
```python
# FlowTemplateVersion query:
result = await db.execute(select(FlowTemplateVersion).where(...))

templates = result.scalars().all()
```
