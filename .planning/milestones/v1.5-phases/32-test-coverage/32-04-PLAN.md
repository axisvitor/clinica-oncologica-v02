# Plan 32-04: Shim Contract Tests Verifying v1.3 Export Parity

## Goal

Create a single test module that verifies symbol-set parity for all 6 v1.3 shim files. Each shim has an `__all__` list declaring its re-exported symbols. The tests assert that every symbol in the shim's `__all__` is actually importable from the shim module, and that the shim's `__all__` matches the expected symbol set documented in the shim file itself. This catches v1.3 regressions where a canonical package renames or removes a symbol that the shim still claims to export.

## Must-Haves (goal-backward)

Tracked in YAML frontmatter. Checklist for executor reference:
- [ ] A test for each of the 6 shims verifies that every symbol in the shim's `__all__` is present as an attribute on the shim module (no ImportError, no AttributeError)
- [ ] A test for each of the 6 shims verifies the expected symbol set matches the shim's actual `__all__` — no missing symbols, no unexpected extra symbols
- [ ] The expected symbol sets are defined explicitly in the test (not derived from the shim itself) to catch cases where `__all__` shrinks undetected
- [ ] All 6 shims are covered: `flow_core.py`, `enhanced_flow_engine.py`, `flow_management.py`, `flow_dashboard.py`, `flow_monitoring.py`, `flow_integrity.py`
- [ ] All tests pass: `cd backend-hormonia && python -m pytest tests/unit/services/test_shim_symbol_parity.py -x -q`

<context>
TEST-04 requires contract tests that verify all 6 shim exports match canonical package exports. The existing split contract tests check identity (`shim.X is canonical.X`) but NOT completeness (`set(shim.__all__) == expected_set`). The gap is that a shim could silently lose an export if the canonical package changes its `__all__` or removes a symbol.

The test approach: define the expected symbol set for each shim as a frozen set literal in the test. Import the shim module. Assert `set(shim.__all__) == expected_set`. Also assert that each symbol in `__all__` is actually accessible via `getattr(shim, symbol)`. This catches both:
1. Stale `__all__` entries (symbol in list but not actually importable)
2. Missing `__all__` entries (symbol was removed from list but still needed)
</context>

**Parametrized test class: `TestShimSymbolParity` (TEST-04)**

Use `@pytest.mark.parametrize` with `SHIM_REGISTRY.items()` to generate one test per shim:

1. `test_shim_all_matches_expected_symbol_set(shim_module_name, expected_symbols)` — Import module via `importlib.import_module(shim_module_name)`. Get `actual_all = set(getattr(module, "__all__", []))`. Assert `actual_all == expected_symbols`, with a clear error message showing missing and extra symbols:
```python
missing = expected_symbols - actual_all
extra = actual_all - expected_symbols
assert not missing and not extra, (
    f"{shim_module_name}: missing={missing}, extra={extra}"
)
```

2. `test_shim_all_symbols_are_importable(shim_module_name, expected_symbols)` — Import module. For each symbol in `module.__all__`, assert `hasattr(module, symbol)` is True. This catches stale `__all__` entries where the import line was removed but the name stayed in the list.

3. `test_shim_all_symbols_are_not_none(shim_module_name, expected_symbols)` — Import module. For each symbol in `module.__all__`, assert `getattr(module, symbol) is not None`. This catches cases where an import silently resolves to None (e.g., `from pkg import X` where X is defined as `X = None` in pkg).

Implementation pattern for parametrize:
```python
@pytest.mark.parametrize(
    "shim_module_name,expected_symbols",
    list(SHIM_REGISTRY.items()),
    ids=list(SHIM_REGISTRY.keys()),
)
class TestShimSymbolParity:
    def test_shim_all_matches_expected_symbol_set(self, shim_module_name, expected_symbols):
        module = importlib.import_module(shim_module_name)
        actual_all = set(getattr(module, "__all__", []))
        missing = expected_symbols - actual_all
        extra = actual_all - expected_symbols
        assert not missing and not extra, (
            f"{shim_module_name}: missing={missing}, extra={extra}"
        )

    def test_shim_all_symbols_are_importable(self, shim_module_name, expected_symbols):
        module = importlib.import_module(shim_module_name)
        for symbol in module.__all__:
            assert hasattr(module, symbol), (
                f"{shim_module_name}.{symbol} in __all__ but not importable"
            )

    def test_shim_all_symbols_are_not_none(self, shim_module_name, expected_symbols):
        module = importlib.import_module(shim_module_name)
        for symbol in module.__all__:
            assert getattr(module, symbol) is not None, (
                f"{shim_module_name}.{symbol} resolved to None"
            )
```
