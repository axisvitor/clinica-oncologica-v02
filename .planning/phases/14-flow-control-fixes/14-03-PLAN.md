       ```python
       from app.models.message import Message, MessageStatus, MessageDirection
       pending_messages = (
           self.db.query(Message)
           .filter(
               Message.patient_id == patient_id,
               Message.status.in_([MessageStatus.PENDING, MessageStatus.SCHEDULED, MessageStatus.QUEUED]),
               Message.direction == MessageDirection.OUTBOUND,
           )
           .all()
       )
       ```
    4. For each pending message, set `message.status = MessageStatus.CANCELLED` (use the existing CANCELLED enum value — check if it exists, if not use "failed" and note in message_metadata).
    5. **Revoke queued Celery tasks** for pending messages. Use the existing codebase pattern from `app/api/v2/routers/tasks/utils/celery_integration.py`:
       ```python
       from celery.result import AsyncResult
       from app.celery_app import celery_app as celery_instance
       revoked_count = 0
       for msg in pending_messages:
           if msg.message_metadata and msg.message_metadata.get("celery_task_id"):
               try:
                   AsyncResult(msg.message_metadata["celery_task_id"], app=celery_instance).revoke(terminate=False)
                   revoked_count += 1
               except Exception as e:
                   logger.warning(f"Failed to revoke Celery task for message {msg.id}: {e}")
       ```
       Note: Use `terminate=False` to avoid killing running workers; just prevent future execution.
    6. **Update flow state:**
       - `flow_state.status = "cancelled"`
       - `flow_state.completed_at = now_sao_paulo()`
       - `flow_state.state_data = flow_state.state_data or {}`
       - `flow_state.state_data["paused"] = False`  (clear any pause state)
       - `flow_state.state_data["cancelled"] = True`
       - `flow_state.state_data["cancelled_at"] = now_sao_paulo().isoformat()`
       - `flow_state.state_data["cancelled_by"] = str(user_id) if user_id else None`
       - `flow_state.state_data["messages_cancelled"] = len(pending_messages)`
       - `flow_state.state_data["tasks_revoked"] = revoked_count`
       - Pop `auto_resume_at` from state_data if present.
    7. **Commit** with version increment (same sync commit pattern as pause/resume):

       ```python
       expected_version = flow_state.version

       self.db.commit()
       ```

       ```python
           "Flow cancelled",
           extra={
       ```
