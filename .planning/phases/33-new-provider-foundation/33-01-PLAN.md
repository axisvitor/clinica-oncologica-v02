WuzAPI text send contract:
- Endpoint: POST /chat/send/text
- Request: {"Phone": "5511987654321", "Body": "Message text"}
- Headers: {"Content-Type": "application/json", "Authorization": "{token}"}
- Response: {"code": 200, "data": {"Id": "ABC123", "Details": "Sent", "Timestamp": "..."}, "success": true}
- Message ID at: response["data"]["Id"]
