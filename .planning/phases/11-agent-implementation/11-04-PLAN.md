**Step 3: Wire GeminiDomainClient shim (client_domain.py)**

Add a private helper method to `GeminiDomainClient`:
```python
def _use_pydantic_agents(self) -> bool:
    """Check if pydantic-ai agents should be used instead of direct generate_content."""
    from app.config import settings
    ai_framework = getattr(settings, "AI_FRAMEWORK", "legacy")
    if ai_framework != "pydantic-ai":
        logger.info("AI_FRAMEWORK=%s -- using GeminiClient.generate_content() path", ai_framework)
        return False
    return True
```
