<objective>
Close TEST-04 verification gap: add a test that invokes UnifiedWhatsAppService.send_message() with an opted-out patient and asserts the guard blocks the send (returns False).

Purpose: The verifier flagged TEST-04 as PARTIAL because `test_send_guard_blocks_opted_out_patient` in `test_opt_out_lgpd.py` only checks a mock attribute predicate — it never imports or invokes `UnifiedWhatsAppService.send_message()`. This plan adds a new async test that creates a real `UnifiedWhatsAppService` instance, calls `send_message()` with a message whose patient has `messaging_stopped_at` set, and asserts `False` is returned. A companion test confirms `send_message()` proceeds (does not return False) when the patient is not opted out.

Output: Two new test functions in `test_opt_out_lgpd.py`; the existing two tests are preserved.
