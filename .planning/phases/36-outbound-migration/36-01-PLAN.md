Migrate UnifiedWhatsAppService from EvolutionAPIClient to WuzAPIClient for all outbound WhatsApp messaging. After this plan, the direct-send path and queue-service wiring both use WuzAPI. No Evolution client is instantiated.

Purpose: This is the primary caller that routes ALL outbound messages -- both direct API sends and queue-based sends. Swapping this service's client eliminates the Evolution dependency for outbound messaging.
Output: `unified_whatsapp_service.py` fully uses WuzAPIClient with correct phone format, response extraction, health check, and circuit breaker naming.

```python
async def _get_wuzapi_client(self) -> WuzAPIClient:
    """Get WuzAPI client (replaces Evolution _get_queue_client)."""
    if not self._wuzapi_client:
        token = getattr(settings, "WHATSAPP_WUZAPI_TOKEN", None)
        base_url = getattr(settings, "WHATSAPP_WUZAPI_BASE_URL", "")
        if not token:
            raise ExternalServiceError("WuzAPI not configured: WHATSAPP_WUZAPI_TOKEN missing")
        self._wuzapi_client = get_wuzapi_client(base_url=base_url, token=token)
        await self._wuzapi_client.connect()
    return self._wuzapi_client
```
