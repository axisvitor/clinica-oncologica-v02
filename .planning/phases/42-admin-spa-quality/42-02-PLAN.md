<objective>
Migrate AgentSwarm.tsx and SystemHealth.tsx from manual useEffect + setInterval polling loops to TanStack Query's useQuery with refetchInterval.

Purpose: Both components currently manage their own polling lifecycle with setInterval, manual state (loading, error, data), and cleanup. TanStack Query v5 (already installed at 5.62.0) handles all of this declaratively with automatic background refetch, deduplication, window-focus refetch, and proper cleanup. This eliminates boilerplate and prevents stale-data bugs.

Output: Both components use useQuery({refetchInterval: 30000}) instead of raw polling; same UI preserved.
</objective>
